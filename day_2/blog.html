Hướng dẫn xây dựng ứng dụng Todo List bằng HTML, CSS và JavaScript

📝 Giới thiệu

Ứng dụng Todo List là một dự án tuyệt vời để bắt đầu học JavaScript thuần (Vanilla JS). Dự án này giúp bạn hiểu cách thao tác DOM, xử lý sự kiện và làm việc với mảng.

Chúng ta sẽ xây dựng một giao diện đơn giản, nơi người dùng có thể:

  • Nhập nội dung công việc
  • Thêm công việc vào danh sách
  • Xóa công việc khỏi danh sách

📁 Bước 1: Tạo cấu trúc HTML

Chúng ta sẽ bắt đầu bằng việc tạo khung giao diện cơ bản.

Mô tả

HTML sẽ bao gồm:

  • Một ô nhập để người dùng điền nội dung công việc
  • Một nút để thêm công việc
  • Một danh sách để hiển thị các công việc

Code HTML

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Todo List App</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <div class="todo-container">
    <h1>My Todo List</h1>
    <div class="input-group">
      <input type="text" id="todo-input" placeholder="Nhập công việc...">
      <button id="add-button">Thêm</button>
    </div>
    <ul id="todo-list"></ul>
  </div>

  <script src="script.js"></script>
</body>
</html>

Giải thích

  • #todo-input: nơi người dùng nhập công việc
  • #add-button: nút bấm để thêm công việc
  • #todo-list: danh sách công việc sẽ được hiển thị tại đây

🧩 Bước 3: Xử lý logic với JavaScript

Mô tả

JavaScript sẽ:

  1. Lấy nội dung người dùng nhập
  2. Tạo một thẻ <li> mới
  3. Thêm nội dung vào danh sách
  4. Thêm nút xóa để xóa từng công việc

Code JavaScript (script.js): 

// Lấy các phần tử cần thao tác
const input = document.getElementById('todo-input');
const addButton = document.getElementById('add-button');
const todoList = document.getElementById('todo-list');

// Gắn sự kiện khi bấm nút "Thêm"
addButton.addEventListener('click', function () {
  const task = input.value.trim();

  if (task !== '') {
    // Tạo thẻ li mới
    const li = document.createElement('li');
    li.textContent = task;

    // Tạo nút xóa
    const deleteBtn = document.createElement('button');
    deleteBtn.textContent = 'Xóa';
    deleteBtn.classList.add('delete-button');

    // Gắn sự kiện xóa khi bấm nút
    deleteBtn.addEventListener('click', function () {
      todoList.removeChild(li);
    });

    // Gắn nút xóa vào li
    li.appendChild(deleteBtn);

    // Thêm li vào danh sách
    todoList.appendChild(li);

    // Xóa nội dung trong ô input
    input.value = '';
  } else {
    alert('Vui lòng nhập công việc!');
  }
});

Giải thích

  • .trim() loại bỏ khoảng trắng đầu/cuối
  • createElement để tạo thẻ HTML mới
  • appendChild để thêm phần tử con
  • Gắn sự kiện click vào nút xóa để loại bỏ từng công việc khỏi danh sách

🔍 Kết quả

Khi bạn hoàn thành cả 3 file (HTML, CSS, JS), mở file index.html bằng trình duyệt, bạn sẽ thấy ứng dụng Todo List đơn giản đã sẵn sàng hoạt động:

  • Nhập công việc → Bấm “Thêm” → Công việc xuất hiện
  • Bấm “Xóa” → Công việc bị loại khỏi danh sách

  • Thêm, hiển thị và xóa công việc trong danh sách
